## Getting Started

Esto es un ejemplo de como puedes ejecutar las ordenes para configurar el repositorio de manera local.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Vicdrilo/S8-BalanceGastos.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```
